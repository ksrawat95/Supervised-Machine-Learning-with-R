# Supervised-Machine-Learning-with-R

## Project Overview
This repository contains a collection of R scripts implementing supervised machine learning algorithms. The project utilizes various datasets, such as customer churn data (`Customer-Churn.csv`) and diabetes data (`diabetes.csv`), to demonstrate linear regression, logistic regression, decision trees, and random forests in R.

## Tech Stack
- **Programming Language**: R
- **Key Packages**:
  - Regression and Classification: `caTools`, `rpart`, `randomForest`
  - Performance Evaluation: `ROCR`

## Key Implementation Details
- **Linear Regression (`LINEAR REGRESSION WITH ONE OR MULTIPLE VARIABLES.R`)**: Implements simple and multiple linear regression models to predict continuous dependent variables.
- **Logistic Regression (`LOGISTIC REGRESSION IN R.R`)**: Builds classification models to predict binary categorical outcomes, such as customer churn and diabetes risk, and evaluates them using confusion matrices.
- **Decision Trees (`DECISION TREE IN R.R`)**: Builds decision trees using the `rpart` package for intuitive rules-based classification.
- **Random Forest (`RANDOM FOREST.R`)**: Fits ensemble random forest classifiers using the `randomForest` package to improve model accuracy and stability.
